# ACG
ACG group work

## Requirements
- **Node.js**
- **npm** (Comes with Node.js)

If not installed, please install Node.js via [nodejs.org](https://nodejs.org/ja), Homebrew (`brew install node`), or your system's package manager.

## How to Run
1. Install dependencies:

   ```bash
   npm install
   ```
1. Start the development server:

   ```bash
   npm run dev
   ```
1. Open the application: navigate to the URL shown in your terminal (usually `http://localhost:5173`).
